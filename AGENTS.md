# Agent instructions

## Language convention (this repository)

- **Repository files** (`CONTEXT.md`, PRDs under `.scratch/`, `docs/**`, ADRs, issue markdown, code comments meant for maintainers): write in **English**.
- **Product UI copy** in apps: use **Spanish** (and keep translation/i18n patterns consistent with each app’s approach).
- **Grill sessions** (`/grill-me`, `/grill-with-docs`): you may ask the maintainer questions in **Spanish** for clarity; anything **saved to the repo** from those sessions (glossary updates, PRDs, ADRs, specs) must be **English**.

## Agent skills

### Issue tracker

Issues and PRDs live as markdown under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five triage roles use the default strings (`needs-triage`, …). See `docs/agents/triage-labels.md`.

### Domain docs

**Single-context** layout: `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
